# MovieHub
**MovieHub** — это HTTP-сервер на Go для агрегации данных о фильмах из открытых источников (Кинопоиск и TMDB). Сервис объединяет информацию из разных API, предоставляя единый интерфейс для получения данных о фильмах с кэшированием в PostgreSQL.

## 🎯 О проекте

Проект решает проблему фрагментированности информации о фильмах, агрегируя данные из нескольких источников в единый формат. Приоритетным источником является **Кинопоиск API**, TMDB используется для обогащения данных.

**Основные возможности:**
- Поиск фильмов по названию (краткая информация)
- Получение расширенных данных по ID с агрегацией из двух источников
- Кэширование результатов в PostgreSQL
- Логирование запросов в Redis

## ✨ Функциональность

### 1. Поиск фильмов по названию
GET /api/movies/search?query=Title

Поиск выполняется только в Кинопоиске. Возвращает список фильмов с краткой информацией:
- ID Кинопоиска
- Название на русском и английском
- Тип фильма
- Год релиза
- Жанры
- Рейтинги

### 2. Получение расширенной информации по ID
GET /api/movies/{id}
Алгоритм работы:
1. Проверка наличия фильма в PostgreSQL по ID
2. Если найден — возврат полных данных из БД
3. Если не найден — последовательные запросы в Кинопоиск и TMDB
4. Агрегация данных в единую структуру
5. Сохранение в БД для будущих запросов

### 3. Список сохраненных фильмов
GET /api/movies/list
Возвращает все фильмы из базы данных с полной информацией.

## 🛠 Технический стек

|-----------|------------|------------|
| **Язык** | Go 1.25.1 | Основной язык разработки |
| **База данных** | PostgreSQL 16 | Хранение данных о фильмах |
| **Кэш/логи** | Redis 7 | Логирование запросов |
| **Контейнеризация** | Docker | Запуск инфраструктуры |
| **API клиенты** | HTTP Client | Взаимодействие с внешними API |

## 📊 Схема базы данных

```sql
movies - основная информация о фильмах
ratings - рейтинги фильмов
genres - справочник жанров
movie_genres - связь фильмов и жанров
casts - актерский состав

Связи между таблицами:

ratings.movie_id → movies.id

movie_genres.movie_id → movies.id

movie_genres.genre_id → genres.id

casts.movie_id → movies.id
```
