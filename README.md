# MovieHub
Приложение для получения информации о фильмах и сериалах из разных открытых источников. Информация представляет собой агрегированые данные о запрашиваемом фильме/сериале, как в укороченном виде, так и в развернутом. 
#Быстрый запуск

Реализовать пулл api для получения информации о фильмах и сериалах из двух источников данных (далее просто источники):

-TMDB 
-KinoPoisk Unofficial API
Запросы к источникам выполнять одновременно.
