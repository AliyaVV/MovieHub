>  Техническое задание
Цель проекта: http-сервер для работы с данными по фильмах из нескольких внешних источников(Кинопоиск и TMDB)

Задачи:
1. Реализовать пулл api для получения информации о фильмах:
   * GET /api/movies/search?query=TitleMoovie - поиск фильма по названию и вывод краткой информации о нем
   * GET /api/movies/{id} - поиск фильма по ID кинопоиска и вывод расширенных данных по запрашиваемому фильму
   * GET /api/movies/list - получение списка сохраненных фильмов из БД

    Источники данных:
    * TMDB 
    * KinoPoisk Unofficial API - приоритетный источник данных
  
    API источников:
    * TMDB  GET https://api.themoviedb.org/3/search/movie?query=Inception&api_key=YOUR_KEY (документация https://developer.themoviedb.org/reference/intro/getting-started)
    * KinoPoisk https://api.kinopoisk.dev/v1.4/movie/search (документация https://api.kinopoisk.dev/documentation#/)
  
Приоритетным источником данных является KinoPoisk, те при ошибках запроса кинопоиска возвращать ошибку и прекращать дальнейшую обработку.
2. GET /api/movies/search?query=TitleMoovie - поиск по названию представляет собой список фильмов с краткой информацией по каждому из них. Запрос происходит только в Кинопоиск.
В ответе должны быть следюущие поля:
Id кинопоиска, название на русском, название на английском, тип фильма, год релиза, жанры, рейтинги
3. GET /api/movies/{id} - поиск по ID кинопоиска представляет собой  агрегированную информацию по запрашиваемому фильму. Сначала запрос проверяет по переданному ID фильм в Postgres, если находит, то возарщает полные данные из всех таблиц(5), если в БД данных нет, тогда запрос выполняется в Кинопоиск и TMDB. Полученная информация из обоих источников парсится и агрегируется в один общий расширенный ответ. Агрегированный данные сохраняются в БД.
4. Реализовать логирование запросов в Redis.
5. GET /api/movies/list - получение всех сохраненных фильмов из Postgres. Список представляет собой полную информацию о фильмах.
6. Таблицы Postgres:
movies - таблица с основной информацией по фильмам
ratings - таблица с рейтингами фильмов
genres - таблица-справочник с возможными жанрами фильмов
movie_genres - таблица пересечения фильмов и жанров
casts - таблица с актерским составом фильмов
5. Логировать все запросы в Redis.
